# Support

## Getting Help

- **Bugs** — Open a [GitHub Issue](https://github.com/groupesti/laravel-ca-crl/issues). Use the bug report template and include your PHP version (8.4+), Laravel version (12.x or 13.x), and package version.
- **Questions & Discussion** — Use [GitHub Discussions](https://github.com/groupesti/laravel-ca-crl/discussions) for general questions, ideas, and community support.
- **Security Vulnerabilities** — Do **not** open a public issue. See [SECURITY.md](SECURITY.md) for reporting instructions.

## Before Opening an Issue

1. Search existing issues and discussions to avoid duplicates.
2. Include a minimal reproduction (code snippet or repository link).
3. Specify your environment: PHP version, Laravel version, package version, and OS.

## What GitHub Issues Are For

GitHub Issues are reserved for **confirmed bugs** and **actionable feature requests**. They are not a general support forum. For usage questions, please use GitHub Discussions.
